# 微信克隆版 - 社交软件

这是一个使用HTML、CSS和JavaScript创建的类似微信的社交软件界面。

## 功能特性

### 🎯 核心功能
- **用户认证系统**：登录和注册功能
- **聊天界面**：实时消息发送和接收
- **联系人管理**：查看和管理联系人列表
- **多标签导航**：微信、通讯录、发现、我 四个主要页面

### 📱 界面特色
- **响应式设计**：适配不同屏幕尺寸
- **微信风格UI**：绿色主题，符合微信设计语言
- **流畅动画**：页面切换和交互动画效果
- **模拟数据**：内置示例联系人和聊天记录

### 💬 聊天功能
- 发送和接收消息
- 消息时间戳显示
- 自动回复模拟
- 聊天记录保存

## 快速开始

1. **下载项目**：将所有文件保存到本地目录
2. **打开应用**：用浏览器打开 `index.html` 文件
3. **开始使用**：
   - 使用任意用户名和密码登录（演示用途）
   - 或点击"注册新账号"创建账户

## 使用说明

### 登录页面
- 输入用户名和密码
- 点击"登录"进入主界面
- 点击"注册新账号"切换到注册页面

### 主界面导航
- **微信**：查看聊天列表，点击联系人开始聊天
- **通讯录**：查看所有联系人
- **发现**：查看朋友圈、扫一扫等功能入口
- **我**：查看个人资料和设置选项

### 聊天功能
- 在聊天列表点击联系人进入聊天界面
- 输入消息后点击发送按钮或按回车键
- 对方会自动回复模拟消息
- 点击左上角返回按钮回到主界面

## 技术实现

### 前端技术栈
- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript (ES6+)**：交互逻辑
- **Font Awesome**：图标库

### 主要特性
- **单页面应用(SPA)**：无刷新页面切换
- **本地状态管理**：使用JavaScript对象管理应用状态
- **事件驱动**：所有交互通过事件监听器处理
- **模拟数据**：内置示例数据展示功能

## 自定义和扩展

### 添加新联系人
在 `script.js` 文件的 `appState.contacts` 数组中添加新联系人：

```javascript
{
    id: 9,
    name: '新联系人',
    avatar: '👤',
    lastMessage: '开始聊天',
    time: '刚刚'
}
```

### 修改样式
编辑 `css/style.css` 文件来自定义颜色、字体和布局。

### 添加新功能
在 `script.js` 中添加新的函数和事件监听器来扩展功能。
